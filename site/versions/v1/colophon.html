This site is built with the amazing Python framework Flask and with the awesome templating engine Jinja. The web app is hosted with a WSGI server over at PythonAnywhere.
I know that I could just probably really easily use an SSG, but I just want to be able to do ridiculous things :D. So all of the pages are simply self-written HTML, which kinda works.

All the source code of this site (including my blog, ProLi (my custom to-do list), GradeCheck and just some random APIs) is available on GitHub. All the source code and all original other content is available in the public domain, because I don't want to restrict how others can use my code, and I really don't want to deal with the mess that is copyright law.

This site uses a really simple custom theme. The main design of this site was inspired by this Jekyll theme, and several page layouts were inspired by this site (by my friend Robin).
